# Strombecker Slot Car Track Layout Tool
## Overview
I created this layout tool with Gemini to help me determine which layout works well for me based on the space I have available.
## Import/Export
- You can import or export a layout, S=straight, L=Left Curve, R=Right Curve, X= Cross track.
